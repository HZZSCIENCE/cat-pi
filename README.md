# 🐱 Cat-Pi / 猫派

一行命令安装 pi + Armory 全家桶 / One-command pi + Armory install.

```bash
npm install -g @tropical_meow/cat-pi
cat-pi
```

## 包含 / What's Included

| 功能 / Feature | 命令 / Command | 默认 / Default |
|---------|---------|---------|
| pi 编程代理 / pi agent | `cat-pi` | - |
| 会话树预览 / Tree Preview | `/armory` | [默认开启 · Default ON] |
| 计划模式 / Plan Mode | `/plan` | 手动 / manual |
| 回收站 / Trash | `/trash` | [默认开启 · Default ON] |
| PLAN.md 自动同步 / Auto-sync | 自动 / auto | ✅ |
| 启动横幅 / Startup Banner | - | ✅ |

## 为什么用 Cat-Pi / Why

原来需要 / Instead of:
```bash
npm install -g @mariozechner/pi-coding-agent
pi install git:github.com/HZZSCIENCE/pi-armory
/armory on
```

现在只需 / Just:
```bash
npm install -g @tropical_meow/cat-pi
cat-pi
```

开箱即用 / Everything works out of the box.

## 升级 / Update

```bash
/cat-update
```

或 / or:

```bash
npm update -g @tropical_meow/cat-pi
```
