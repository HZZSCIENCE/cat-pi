# 🐱 Cat-Pi

One command to install pi coding agent with all Armory plugins.

```bash
npm install -g cat-pi
cat-pi
```

## What's included

| Feature | Command | Default |
|---------|---------|---------|
| pi coding agent | `cat-pi` | - |
| Session Tree Preview | `/armory` | ✅ ON |
| Plan Mode | `/plan` | manual |
| Trash Recycle Bin | `/trash` | ✅ ON |
| PLAN.md auto-sync | automatic | ✅ |

## Why Cat-Pi?

Instead of:
```bash
npm install -g @mariozechner/pi-coding-agent
pi install git:github.com/HZZSCIENCE/pi-armory
/armory install
```

Just:
```bash
npm install -g cat-pi
cat-pi
```

Everything works out of the box.
